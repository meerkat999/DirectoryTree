## Instructions:

- Install:

  - npm install
  - Set the enviroment variable FILE_PATH with the path of the input file (default = "<project_folder>/input.txt")

- Use (expect the output on the node console):

  - npm start

- Test:

  - npm test

## Tech Stack:

- Test: Mocha, Tai
- NodeJS & Typescript
